# 📊 Ferramenta de Controle de Investimentos com Excel | Plataforma DIO
Este projeto foi desenvolvido como parte de um desafio prático incluso no curso Santander - Excel com Inteligencia Artificial, com o objetivo de criar uma ferramenta em Excel que auxiliasse no controle de investimentos, organização financeira e análise de dados.

Essa ferramenta se desenvolveu ao responder 5 peguntas básicas: 
- Quanto investir por mes?
- Por quantos anos?
- Qual a taxa de rendimento mensal?
- Quanto há de patrimonio acumulado?
- Há dividendos mensais?

### O que foi desenvolvido?
Com o auxilio do professor Felipe e as ferramentas disponiveis dentro da plataforma (Excel), pude desenvolver uma planilha automatizada para o controle de investimentos, organizando dados financeiros e utilizando formulas como PROCV E VF que visa calcular o valor futuro de investimento com base em uma taxa de juros constante. 

## 🧠 O que eu aprendi?

- Organização de dados no Excel
- Uso de fórmulas financeiras
- Estruturação de dashboards
- Boas práticas na documentação de projetos
- Publicação de projetos no GitHub
